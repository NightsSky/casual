# Screen Inventory

## 导航结构

| 模块 | 路由 | 移动端导航 | 桌面端导航 | 说明 |
| --- | --- | --- | --- | --- |
| 笔记 | `/notes` | 底部 Tab | 左侧栏 | 默认首页 |
| 编辑器 | `/notes/:noteId` | 独立页面，隐藏底部 Tab | 与笔记列表分栏 | 支持编辑/预览 |
| 助手 | `/reminder` | 底部 Tab | 左侧栏 | 提醒列表与弹窗 |
| 计划 | `/plan` | 底部 Tab | 左侧栏 | 当前为空白页面 |
| 设置 | `/settings` | 底部 Tab | 左侧栏 | 仓库/平台配置入口、同步设置、关于 |
| 仓库管理 | `/settings/repository` | 设置内二级页 | 设置内二级页 | Git 同步状态和操作 |
| Git 平台配置 | `/settings/platform-config` | 设置内二级页 | 设置内二级页 | GitHub/Gitee 与仓库连接信息 |
| Token 帮助 | `/settings/platform-config/token-help` | 平台配置内二级页 | 平台配置内二级页 | GitHub/Gitee Token 获取说明 |

## 响应式布局

### 移动端

- 底部四 Tab：笔记、助手、计划、设置。
- 编辑器页面隐藏底部 Tab，顶部返回。
- 笔记新建入口是右下角 FAB，展开后选择 TXT / Markdown。
- 提醒页移动端使用 AppBar 的加号创建提醒。

### 桌面端

- 左侧 240px 侧边栏，主区域展示当前模块。
- 笔记模块在选中笔记后进入二栏布局：左侧列表，右侧编辑器。
- 笔记可拖出为独立窗口，仅 Windows 桌面支持。
- 提醒页使用 FAB 创建提醒。

## 页面清单

### 1. 笔记列表

主要元素：
- 顶部标题 `casual`
- 新建笔记入口：桌面端顶部加号，移动端 FAB
- 同步按钮
- 排序菜单：最近更新、创建时间、标题排序
- 同步状态条：同步中、上次同步、同步失败
- 标签筛选条：全部、具体标签
- 笔记卡片：标题、摘要、标签、更新时间、同步状态

必须覆盖状态：
- 空状态：还没有笔记
- 普通列表
- 本地未同步标签
- 冲突标签
- 同步失败横条
- 已在独立窗口打开的角标
- 长按/右键菜单：编辑、置顶、删除、在新窗口打开

### 2. 编辑器

主要元素：
- 顶部返回按钮
- 当前笔记标题
- 编辑/预览切换按钮
- 更多菜单：同步到远程、管理标签、导出 Markdown、格式转换、删除笔记
- 标题输入或预览标题
- 标签行：标签 Chip、添加标签入口
- 内容编辑区或 Markdown 预览区
- Markdown 工具条：标题、加粗、列表、引用、代码等
- 底部字数和更新时间

必须覆盖状态：
- 新建笔记编辑态
- 已有笔记预览态
- Markdown 编辑态
- TXT 编辑态
- 添加标签弹窗
- 删除确认弹窗
- 同步中弹窗
- 同步失败 Snackbar
- 独立窗口编辑时只读横幅

### 3. 助手/提醒

主要元素：
- 提醒列表
- 提醒卡片：标题、时间/间隔、重复规则、启用开关
- 新建/编辑提醒弹窗
- 重复方式：无、每天、每周、每月、工作日、按间隔
- 间隔选择：小时、分钟

必须覆盖状态：
- 空状态
- 提醒列表
- 停用提醒
- 新建提醒弹窗
- 编辑提醒弹窗
- 按间隔提醒设置
- 删除提醒确认
- 保存失败提示

### 4. 计划

主要元素：
- 页面标题、说明与“新建计划”主操作
- 状态筛选：全部、进行中、已逾期、已完成、已终止
- 计划卡片：标题、目标摘要、整体状态、自动进度、完成步数、下一步、最终截止时间
- 计划概览：开始时间、最终截止时间、下一步和整体剩余时间
- 步骤时间轴：步骤序号、标题、预计时间、提醒、局部状态、完成时间和完成说明
- 步骤操作：完成步骤、撤销完成，允许跳步操作
- 执行动态：创建、信息修改、步骤修改、步骤完成、撤销、自由记录和终止
- 新建/编辑表单：标题、目标、开始时间、有序步骤列表和每步独立提醒

响应式规则：
- 窄屏为列表 → 独立详情页，最小宽度 360px 不溢出
- 宽屏为左侧计划列表 + 右侧步骤时间轴/执行动态双栏
- 步骤编辑卡支持拖动排序，保存时校验预计时间非递减

必须覆盖状态：
- 空计划列表
- 单步骤和多步骤
- 步骤待完成、局部逾期、已完成
- 计划未开始、进行中、整体逾期、已完成、已终止
- 无提醒、预设提醒、自定义提醒
- 跳步完成、全部自动完成、撤销完成
- 表单校验失败、持久化失败

### 5. 仓库管理（设置内二级页）

主要元素：
- 顶部标题：仓库管理
- 连接状态卡片：已连接/未连接，平台/owner/repo
- 快捷操作：拉取远程、推送本地、全量同步、仓库设置
- 同步统计：总笔记、未同步、已同步、冲突
- 同步记录：成功、失败、警告

必须覆盖状态：
- 未配置仓库
- 已连接
- 未连接
- 有冲突
- 空同步记录
- 多条同步记录

### 6. 设置总览

主要元素：
- 仓库管理入口
- Git 平台配置入口
- 同步设置：自动同步、编辑时自动推送
- Windows 窗口设置：关闭主面板时
- 关于：版本、清除所有数据

必须覆盖状态：
- 关闭行为选择底部弹窗
- 清空所有数据确认

### 7. Git 平台配置

主要元素：
- GitHub/Gitee、Access Token、用户名/组织、仓库名、分支、笔记目录
- Token 帮助按钮
- 测试连接按钮
- 保存配置按钮

必须覆盖状态：
- Token 隐藏输入
- 平台选择底部弹窗
- 连接成功/失败 Snackbar

### 8. Token 帮助
主要元素：
- 返回按钮
- GitHub Token 获取步骤
- Gitee Token 获取步骤
- Token 安全说明
- 官方入口链接文本

必须覆盖状态：
- 移动端单列
- 桌面端 760px 内容宽度
